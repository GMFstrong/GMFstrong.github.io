Docker网络的使用

other
  view

1) 同一宿主机的容器互联：

默认容器的IP是DHCP自动分配的，因此在容器间相互访问时不太方便（需要进入容器查看IP,并且在容器销毁重建以后IP地址会发生变化），故而通常采用容器名称或者别名来实现容器之间的相互访问。

3) docker跨主机通信：

默认宿主机的 docker0 网卡都是172.17.0.x/24网段，故而要实现容器跨主机之间的通信势必先更改Docker的默认网段，使其不在同一网络。

4) 自定义网络：

docker 的自定义网络可以自定义 IP 地范围和网关。
