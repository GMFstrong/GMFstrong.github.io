Linux下的桥接网卡配置

network
  view

（1）桥设备不支持NetworkManager服务，故必须把该服务关掉。
chkconfig –list NetworkManager
（2）改为network服务
chkconfig –list network
(3)安装桥设备管理工具。
yum install bridge-utils
rpm -ql bridge-utils 查看安装了哪些包及安装位置。
(4)运用brctl创建桥设备：
（1）添加桥接网卡：brctl addbr br0
(2)拆除原来网卡的ip地址：ifconfig ens33 0 up
(3)将br0桥接到ens33：brctl addif br0 ens33
(4)添加br0的ip地址：ifconfig br0 xxx.xxx.xxx.xxx/24 up
(5)添加网关：route add default gw 192.168.1.1
可以通过shell脚本，添加桥设备，防止断网。
通过修改配置文件来配置桥设备(永久生效)：

(1)cd /etc/syscoonfig/network-scripts

(2)vim ifcfg-br0

(3)vim ifcfg-eth0

(4)vim ifcfg-eth1
