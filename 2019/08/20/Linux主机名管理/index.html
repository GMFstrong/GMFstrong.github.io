Linux主机名管理

linux

关于Linux主机名的修改：
hostname 主机名 再执行exec bash可以临时生效。

若要永久更改主机名这需要更改文件/etc/sysconfig/network（在Centos6中），若是在Centos7中，建议更改/etc/hostname，更改配置文件不会立即生效。

如若修改了/etc/sysconfig/network文件，建议在/etc/hosts文件里127.0.0.1后加入自己更改的主机名，便于自己在访问主机名时可以解析。

没有添加以前：

添加以后：

在Centos7中可以使用如下命令管理主机名：
hostnamectl set-hostname 主机名

以上命令用于修改主机名。

默认删除文件/etc/hostname，主机名会恢复至localhost.localdomain。
